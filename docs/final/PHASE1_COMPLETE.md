## ⚠️ ВАЖНЫЕ ПРИМЕЧАНИЯ

1. **Миграция 015 не применена автоматически** — нужно выполнить вручную:
   ```sql
   psql -d vapeshop < db/migrations/015_critical_security_fixes.sql
   ```

2. **Все SELECT запросы должны учитывать `is_active`** — при добавлении новых фич:
   ```sql
   WHERE is_active = true
   ```

3. **Telegram bot требует правильной конфигурации**:
   - Установить вебхук: `https://yourdomain.com/api/bot`
   - Переменная окружения `TELEGRAM_BOT_TOKEN` должна быть установлена

4. **HMAC-верификация требует отправки initData с frontend**:
   - Использовать `window.Telegram.WebApp.initData`
   - Отправить в заголовке `Authorization: tg_init_data <initData>`
